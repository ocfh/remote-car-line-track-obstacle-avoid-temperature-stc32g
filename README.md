# 遥控循迹避障测温智能小车
🚘 基于STC32G12K128单片机开发的差速智能小车，集成HC-SR04超声波自动避障、红外循迹、DS18B20温度采集、串口指令控制、OLED屏幕实时显示、PWM电机调速、灯光与蜂鸣器驱动，代码模块化分层设计，外设驱动独立封装。

## 项目简介
本项目主控为STC32G12K128最小系统板，搭载三轮差速小车底盘，支持前进、后退、左右转向、多档位调速；外设集成HC-SR04超声波测距模块、红外循迹传感器、DS18B20单总线温度传感器、IIC OLED显示屏、串口通信。
全部外设驱动独立封装 `.c/.h`，代码低耦合，内置传感器读取超时保护，防止程序卡死；支持安卓蓝牙串口APP无线遥控、超声波自动避障模式、红外循迹模式三种工作模式自由切换。
配套安卓SPP蓝牙控制APP开源仓库：https://github.com/ocfh/spp-car-controller-android-java ，可搭配本工程实现手机无线下发小车控制指令。

## 核心功能特性
### 1. UART1串口指令遥控
串口中断实时接收APP下发单字节指令，实时响应小车运动、灯光、蜂鸣器、模式切换命令。
- `0x01`：小车前进
- `0x02`：小车停止
- `0x03`：小车后退
- `0x04`：小车右转
- `0x05`：小车左转
- `0x06`：加速档位上调
- `0x07`：减速档位下调
- `0x08`：开启/关闭超声波避障模式
- `0x09`：开启/关闭红外循迹模式
- `0x10`：左转向灯切换
- `0x11`：右转向灯切换
- `0x12`：开启蜂鸣器
- `0x13`：关闭蜂鸣器
- `0x14`：刷新传感器ADC数据

### 2. HC-SR04 超声波自动避障模式
- 距离阈值可自定义，根据障碍物距离分级执行避障策略
- 障碍物较远：保持直行；障碍物中等距离：后退+转向绕行；障碍物过近持续后退报警
- 非阻塞循环测距，增加读取超时判断，避免传感器异常导致程序卡死
- 全局变量 `csb_toggle` 一键开启/关闭自动避障功能

### 3. 红外循迹行驶模式
两路红外循迹传感器采集地面黑白轨迹信号，采用状态机逻辑：
- 两路均检测黑线：保持直行
- 左侧检测黑线、右侧无黑线：小幅左转修正路线
- 右侧检测黑线、左侧无黑线：小幅右转修正路线
- 内置循环纠偏逻辑，防止冲出轨迹，适合简易黑线赛道行驶
- 全局变量 `hwxj_toggle` 一键开启/关闭循迹功能

### 4. DS18B20温度采集
标准单总线时序驱动，稳定读取环境温度；数据解析拆分整数、小数位，直接在OLED屏幕展示；支持周期刷新温度数据。

### 5. IIC-OLED实时数据显示
0.96寸IIC OLED屏幕，循环刷新各项信息：
- ADC电压采集数值
- 当前小车运行状态
- 超声波测距距离
- DS18B20实时温度
- 当前PWM速度档位
- 循迹传感器原始电平状态

### 6. 电机PWM调速系统
定时器生成PWM波形，速度分为5档（20~100），支持逐级加速、逐级减速；封装基础运动函数，方便修改运动逻辑。
```c
//基础运动API
void Car_Forward(void);    //前进
void Car_Backward(void);   //后退
void Car_Left(void);       //左转
void Car_Right(void);      //右转
void Car_Stop(void);       //小车停止
```

### 7. 配套安卓SPP蓝牙控制APP
独立开源安卓蓝牙串口控制软件，仓库地址：https://github.com/ocfh/spp-car-controller-android-java
- SPP经典蓝牙串口通信，适配HC05/HC06蓝牙透传模块
- 虚拟按键控制小车运动、调速
- 功能按键切换避障/循迹模式、控制车灯、蜂鸣器
- 双向通信，可接收下位机上传传感器数据

### 8. 辅助配套功能
- 左右独立转向灯、可控蜂鸣器报警
- 模块化初始化，不需要的外设可注释关闭，节省资源
- 所有传感器读取加入防卡死超时保护
- 多模式互斥逻辑，避障、循迹、手动遥控模式不会冲突

## 快速编译 & 烧录教程
### 开发环境依赖
1. Keil C51
2. STC-ISP下载工具
3. USB转TTL下载器

### 使用步骤
1. 克隆仓库到本地
2. Keil C51打开 `project.uvproj` 工程
3. 根据硬件接线修改引脚宏定义
   - `oled.c` 修改OLED SDA/SCL引脚
   - `motor.h` 修改电机控制引脚
   - `sr04.h` 修改超声波TRIG/ECHO引脚
   - `hw.h` 修改红外循迹传感器引脚
4. 编译工程，无报错后使用STC-ISP烧录程序至STC32G单片机
5. 上电初始化完成，可通过蓝牙APP切换遥控模式、自动避障、循迹模式

### 功能开关配置
1. **关闭超声波避障**：APP发送指令 `0x08` 切换关闭 `csb_toggle`
2. **关闭红外循迹**：APP发送指令 `0x09` 切换关闭 `hwxj_toggle`
3. **调整电机速度档位**：APP发送0x06加速、0x07减速
4. **修改避障距离阈值**：直接修改main.c内距离判断条件

## 关键模块原理说明
### 1. 差速电机PWM控制
Timer4输出PWM信号，通过调整占空比控制电机转速，左右电机独立控制，依靠差速实现转向。速度区间20~100，分为5档调速。

### 2. HC-SR04超声波测距
触发TRIG输出高电平脉冲，等待ECHO回响信号，通过高电平持续时间换算距离，单位cm；主循环定时采集，增加超时处理，防止模块无响应阻塞程序。

### 3. DS18B20单总线通信
严格遵循DS18B20时序，初始化、跳过ROM、读取温度寄存器，拆分温度整数与小数部分，便于OLED屏幕显示。

### 4. 红外循迹检测
TCRT5000红外对管，黑线吸收红外光输出高电平，白纸反射红外光输出低电平；主循环持续读取两路传感器电平，状态机实时调整小车方向。

### 5. IIC OLED屏幕驱动
软件模拟IIC时序，支持8X16英文字符显示，持续刷新多路传感器数据；每次刷新前清屏，界面整洁。

### 6. UART串口通信协议
UART1配置为中断接收，蓝牙模块透传APP下发单字节指令；中断内直接解析指令修改全局状态变量，实时响应控制需求。
